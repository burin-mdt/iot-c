C Client SDK for IBM Watson™ IoT
C Client SDK Return codes

The IoTP_* functions defined in IBM Watson IoT Platform C Client SDK retrurns the following codes:

  • Value of 0 indicates a success response.
  • Value of 1 indicates a failure response.
  • Values of 1001 to 1500 are IOTPRC codes returned by IoTP C Client libraries.
  • Values less than -1 are specific error codes returned by Paho MQTT C client. For details on refer to MQTTReasonCodes in Paho Asynchronous MQTT C Client Library.
