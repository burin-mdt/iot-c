IoTPConfig_create() API creates IoTP client configuration object using the configuration items defined in configuration file (in YAML format) specified by configFileName parameter.

Parameters
config- A pointer to an IoTPConfig handle.
configFileName- Configuration file path. If NULL, the API will return an empty config object.
Returns
IOTPRC - Returns IOTPRC_SUCCESS onsuccess or IOTPRC_* on error
Remarks
Use IoTPConfig_clear() API, to clear all properties.
