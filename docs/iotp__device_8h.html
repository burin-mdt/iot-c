◆ IoTPDevice

typedef void* IoTPDevice

A handle representing a device client of IBM Watson IoT Platform service. A valid client handle is available following a successful call to IoTPDevice_create() API.

The IoTPDevice_create() API creates a device client that can connect to WIoTP service. IBM Watson IoT Platform service. This API takes the following arguments:

◆ IoTPDevice_subscribeToCommands()

DLLExport IOTPRC IoTPDevice_subscribeToCommands ( IoTPDevice *  device,
char *  commandId,
char *  formatString 
)

The IoTPDevice_subscribeToCommands() API subscribe to commands for the device. To receive command specified by "commandId" paramete, a command callback handler must be registered using IoTPDevice_setCommandHandler() API.

Parameters
device- A pointer to IoTP device handle.
commandId- ID of command. The command ID can be any string that is valid in MQTT protocol. Also accepts MQTT wild card character "+".
formatString- Reponse format. e.g. json, xml, txt, csv. Also accepts MQTT wild card character "+"
Returns
IOTPRC - Returns IOTPRC_SUCCESS onsuccess or IOTPRC_* on error
