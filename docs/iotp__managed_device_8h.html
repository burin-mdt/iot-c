◆ IoTPManagedDevice

typedef void* IoTPManagedDevice

A handle representing a device client of IBM Watson IoT Platform service. A valid client handle is available following a successful call to IoTPManagedDevice_create() API.

The IoTPManagedDevice_create() API creates a device client that can connect to IBM Watson IoT Platform service. The device client is created using configuration specified by IoTP Config object.

Parameters

Returns

◆ IoTPManagedDevice_subscribeToCommands()

DLLExport IOTPRC IoTPManagedDevice_subscribeToCommands ( IoTPManagedDevice *  managedDevice,
char *  commandId,
char *  formatString 
)

The IoTPManagedDevice_subscribeToCommands() API subscribe to commands for the device. To receive command specified by "commandId" paramete, a command callback handler must be registered using IoTPManagedDevice_setCommandHandler() API.

Parameters
managedDevice- A pointer to IoTP managed device handle.
commandId- ID of command. The command ID can be any string that is valid in MQTT protocol. Also accepts MQTT wild card character "+".
formatString- Reponse format. e.g. json, xml, txt, csv. Also accepts MQTT wild card character "+"
