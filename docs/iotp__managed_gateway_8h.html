◆ IoTPManagedGateway

typedef void* IoTPManagedGateway

A handle representing a device client of IBM Watson IoT Platform service. A valid client handle is available following a successful call to IoTPManagedGateway_create() API.

The IoTPManagedGateway_create() API creates a device client that can connect to IBM Watson IoT Platform service. The device client is created using configuration specified by IoTP Config object.

Returns
