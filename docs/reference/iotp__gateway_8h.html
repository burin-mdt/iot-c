Detailed Description

Gateway Client Reference exposes functions and declarations to build IoTP gateway client.

Typedef Documentation

◆ IoTPGateway

typedef void* IoTPGateway

A handle representing gateway client of IBM Watson IoT Platform service. A valid client handle is available following a successful call to IoTPGateway_create() API.

◆ IoTPGateway_create()

DLLExport IOTPRC IoTPGateway_create ( IoTPGateway **  gateway,
IoTPConfig *  config 
)

The IoTPGateway_create() API creates a gateway client that can connect to IBM Watson IoT Platform service. The gateway client is created using configuration specified by IoTP Config object.

Parameters
gateway- A pointer in which to return the newly created IoTPGateway handle.
config- A pointer to IoTP config handle.
Returns
IOTPRC - Returns IOTPRC_SUCCESS on success or IOTPRC_* on error
Remarks
Use IoTPGateway_destroy() API, to destroy a gateway client created using this API.

◆ IoTPGateway_subscribeToCommands()

DLLExport IOTPRC IoTPGateway_subscribeToCommands ( IoTPGateway *  gateway,
char *  commandId,
char *  formatString 
)

The IoTPGateway_subscribeToCommands() API subscribes to commands for the gateway. To receive command specified by "commandId" paramete, a command callback handler must be registered using IoTPGateway_setCommandHandler() API.

Parameters
gateway- A pointer to IoTP gateway handle.
commandId- ID of command. The command ID can be any string that is valid in MQTT protocol. Also accepts MQTT wild card character "+".
formatString- Reponse format. e.g. json, xml, txt, csv. Also accepts MQTT wild card character "+"
Returns
IOTPRC - Returns IOTPRC_SUCCESS on success or IOTPRC_* on error
